# FinMath Lab

FinMath Lab is a progressive Python project dedicated to financial mathematics, corporate finance, market finance, econometrics, and simulation.

The objective is to build a personal technical laboratory where each concept is implemented clearly, documented properly, and improved incrementally.

## Main Areas

### 1. Math Core

Basic mathematical tools used in finance and econometrics.

Planned topics:

- Sequences and series
- Matrix operations
- Optimization
- Probability foundations

### 2. Corporate Finance

Tools for investment decisions and financial analysis.

Planned topics:

- Time value of money
- Net Present Value
- Internal Rate of Return
- Annuities
- Bond valuation
- Capital budgeting

### 3. Market Finance

Tools for analyzing financial markets and risk.

Planned topics:

- Returns
- Volatility
- Portfolio analysis
- CAPM
- Value at Risk

### 4. Econometrics

Core econometric tools implemented from scratch or with minimal dependencies.

Planned topics:

- Ordinary Least Squares
- Model diagnostics
- Time series basics
- Forecast evaluation

### 5. Simulations

Simulation-based tools for uncertainty and risk analysis.

Planned topics:

- Monte Carlo simulation
- Random walks
- Stochastic processes
- Scenario analysis

## Project Philosophy

This project will be built progressively through small daily commits.

Each module should contain:

- A clear formula
- A simple implementation
- A numerical example
- Basic documentation
- Tests when relevant
